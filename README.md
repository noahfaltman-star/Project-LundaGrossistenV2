Lundagrossisten:

Detta är ett grupprojekt för kursen Grundläggande programmering. Lundagrossisten är en e-handel för matvaror där användaren kan navigera mellan kategorier, lägga till varor i en dynamisk varukorg och se totalbeloppet uppdateras i realtid.

Projektet är byggt med fokus på Objektorienterad JavaScript och responsiv design.


Funktioner:
Produktvisning: Produkter genereras dynamiskt som kort i ett responsivt rutnät (CSS Grid).
Kategorier: Filtrering av varor (Mejeri, Kött, Frukt, Grönt) via separata sidor.
Dynamisk Varukorg: Lägg till/ta bort produkter.
    Justera antal direkt på produktkortet eller i varukorgen.
    Automatisk uträkning av totalsumma.
Användarfeedback: "Toast"-notifikationer som bekräftar när en vara lagts till eller tagits bort.
Responsivitet: Mobilanpassad meny (hamburgare) och layout som anpassar sig efter skärmstorlek.


Teknisk Stack:
Vi har använt oss av moderna webbstandarder utan externa ramverk:

HTML5: Semantisk struktur med `data-category` för sidhantering.
CSS3:
    Layout: CSS Grid och Flexbox.
    Design: Responsiva @media-queries, animationer och transitions.
JavaScript (ES6+):
    OOP:* Använder `class Product` för att strukturera produktdata och skapa HTML-element.
    DOM-manipulation: För att rendera varukorg och hantera händelser.
Bibliotek/Resurser:
    FontAwesome: För ikoner (varukorg, hamburgarmeny, papperskorg).
    Google Fonts: Typsnittet 'Poppins' och 'Segoe UI'.


Hur man startar projektet

Eftersom projektet består av statiska filer behövs ingen installation av paket.

1 - Ladda ner:
    Klona detta repository eller ladda ner som ZIP-fil.
    
    git clone https://github.com/noahfaltman-star/Project-LundaGrossistenV2.git
    

2 -  Kör:
    Leta upp filen `Home.html` i mappen.
    Öppna filen direkt i din webbläsare (dubbelklicka), eller använd "Live Server" i VS Code för bästa upplevelse.
